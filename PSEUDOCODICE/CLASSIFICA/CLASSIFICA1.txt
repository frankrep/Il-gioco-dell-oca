
aggiornare_classifica_top_10
DATI DI INPUT
file_classifica_top_10, contiene record di tipo giocatore corrispondenti ai dieci migliori giocatori di sempre, file di tipo binario
partita_attuale, la partita da cui leggere la lunghezza del percorso, partita
giocatore_vincitore, vincitore della partita appena conclusa di cui confrontare il numero di lanci per verificarne l'inserimento,vincitore 
DATI DI OUTPUT
file_classifica_top_10, file della classifica eventualmente aggiornato, file di tipo binario
DATI DI LAVORO
inserito, indica se il vincitore è stato già inserito in classifica oppure no, booleano, = VERO (il giocatore è stato inserito all'interno della classifica), = FALSO (il giocatore non è stato inserito in classifica)
indice_vincitori, indice di spostamento all'interno l'array di vincitori, intero, >0
vincitori, elenco dei dieci migliori giocatori di partite , array a una dimensione (NUMERO_MASSIMO_CLASSIFICATI) di GIOCATORE
rateo, rapporto tra i lanci dei dadi e la lunghezza del percorso del vincitore della partita, intero
rateo2, rapporto tra i lanci dei dadi e la lunghezza del percorso del vincitore in posizione indice_vincitori presente già in classifica, intero
ALGORITTMO
vincitori = caricare_classifica (file_classifica_top_10)
inserito = 0
indice_vincitori = 1
rateo= leggere_numero_lanci_vincitore(giocatore_vincitore) / leggere_lunghezza_percorso_vincitore(giocatore_vincitore)
MENTRE ( indice_vincitori <= NUMERO_MASSIMO_CLASSIFICATI )
    rateo2= leggere_lanci_effettuati(elemento in posizione indice_vincitori di vincitori) / leggere_numero_caselle(partita_attuale)
    SE (rateo <= rateo2) ) AND (inserito != 1)
               vincitori = spostamento_destra_vincitori(vincitori,indice_vincitori,giocatore)
               inserito = 1
    FINE
        indice_vincitori = indice_vincitori + 1
FINE

spostamento_destra_vincitori
DATI DI INPUT
vincitori, elenco da spostare verso destra, array di giocatori
posizione_inserimento, posizione all'interno di vincitori in cui inserire il nuovo giocatore, intero, > 0,
giocatore_attuale,giocatore da aggiungere in vincitori,giocatore
DATI DI OUTPUT
vincitori, con giocatore inserito in quella posizione, array giocatore
DATI DI LAVOROIMP
giocatore_salvato, contiene il prossimo giocatore da inserire, giocatore
ALGORITMO
MENTRE (posizione_inserimento <= 10)
        giocatore_salvato = elemento di vincitori in posizione posizione_inserimento
        elemento di vincitori in posizione posizione_inserimento = giocatore_attuale
        giocatore_attuale = giocatore_salvato
        posizione_inserimento = posizione_inserimento + 1
FINE


caricare_classifica 
DATI DI INPUT
file_classifica_top_10, contiene record di tipo giocatore corrispondenti ai dieci migliori giocatori di sempre, file di tipo binario
NUMERO_MASSIMO_CLASSIFICATI, il numero massimo dei giocatori in classifica, Intero, = 10
NOME_FILE_CLASSIFICA_TOP_10, il nome del file contenente la classifica, Array ad una dimensione di caratteri con ultimo carattere FINE_STRINGA
DATI DI OUTPUT
vincitori, i giocatori presenti nel file della classifica, Array ad una dimensione (NUMERO_MASSIMO_CLASSIFICATI) di giocatore
ALGORITMO
vincitori = leggere_file (giocatore, NUMERO_MASSIMO_CLASSIFICATI, NOME_FILE_CLASSIFICA_TOP_10)
